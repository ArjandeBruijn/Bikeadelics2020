Penny Farthings are late 1800s bicycles with a large front wheel and a tiny rear wheel. The large front wheel increases the distance travelled per pedal turn, effectively it is a bicycle gear from a time when the bicycle chain had not been invented yet.
The trip consists of two loops of about 3 and 2 miles respectively. It is all hard pavement and you should rarely have to get off the bike which makes it more suitable for new penny farthing riders. The first loop in green is more scenic and requires few skills. After the first loop there is an option to return to the Jessup farm. The second loop along the spring creek and around the Epic center is more curvy, contains a few bridges and is slightly hilly. A lot of fun for a more ambitious rider!

How it works

Once you're comfortable we take a scenic round tour .

We have mini-penny farthings for 8+ kids and a fun car-bike and bakfiets for kids 4-8. Younger kids can use a bicycle trailer that we can mount behind a penny farthing.
